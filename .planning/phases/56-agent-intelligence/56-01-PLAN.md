```
## 3. 4J HMBC Coupling Detection (Aromatic Systems)

### When to Flag

Flag potential 4J HMBC couplings when ALL of these are true:
1. 4 or more carbons fall in 110-160 ppm range (aromatic expectation)
2. HMBC correlations exist between those aromatic carbons and carbons in 0-55 ppm range (benzylic/alpha-substituent positions)

### Why This Matters

4J HMBC couplings through aromatic rings (W-pathway) are common in para-disubstituted benzenes. LSD enforces HMBC as 2-3 bond, so a 4J correlation silently excludes the correct structure while wrong ring-size isomers survive. These do NOT cause zero solutions -- they cause WRONG solutions.

### Detection Logic

For each HMBC correlation between an aromatic carbon (110-160 ppm) and a non-aromatic carbon (0-55 ppm):
- If the aromatic carbon is a CH (not quaternary), the coupling path goes through the ring
- Flag as "potential 4J" with the atom pair and both shifts
- Note: benzylic CH2/CH next to the ring at 30-55 ppm are the most common 4J partners

### Reporting Format

List flagged correlations in [SETUP-COMPLETE] under "Potential 4J correlations:" section. Use this format:
- `C{N}({shift} ppm, ArCH) <-> H-C{M}({shift} ppm, {mult}): potential 4J through aromatic ring`
- Or "None detected" if no aromatic system or no suspect correlations

Do NOT remove these correlations from the HMBC list. Flag them for the lsd-engineer to defer.
```

```xml
On receiving any structured message (identified by `[TAG]` prefix):
     ```

     ```
4. If all required fields present: proceed to process in monitor_progress

**Important:** "None detected", "N/A", "None", and "0" are all VALID values. Only truly missing field labels trigger resend.
</step>
```
