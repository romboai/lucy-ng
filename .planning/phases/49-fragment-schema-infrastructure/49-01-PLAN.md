Create `src/lucy_ng/fragments/db.py` with a `FragmentDatabaseManager` class following the exact pattern of `database/manager.py`:

**Class structure:**
- `__init__(self, db_path: str | Path)` — stores `Path(db_path)`, initializes `_conn: sqlite3.Connection | None = None`
- `__enter__` / `__exit__` — context manager (connect/close)
- `_connect()` — creates connection with `check_same_thread=False`, `row_factory = sqlite3.Row`, `PRAGMA foreign_keys = ON`
- `connection` property — calls `_connect()`
- `close()` — closes connection, sets `_conn = None`

**Methods:**

1. `create_tables(self) -> None` — executes all `FRAGMENT_SCHEMA_STATEMENTS`, then:
   - `INSERT OR REPLACE INTO schema_meta (key, value) VALUES ('schema_version', str(FRAGMENT_SCHEMA_VERSION))` — always update version
   - `INSERT OR IGNORE INTO schema_meta (key, value) VALUES ('bin_size', '2.0')` — use IGNORE, not REPLACE, to protect existing bin_size on re-runs

2. `get_schema_version(self) -> int | None` — same pattern as `DatabaseManager.get_schema_version()`. Query `schema_meta` for `schema_version` key, return `int(row["value"])` or `None`. Catch `sqlite3.OperationalError` if table doesn't exist.

3. `get_bin_size(self) -> float | None` — query `schema_meta` for `bin_size` key, return `float(row["value"])` or `None`.

4. `get_ssc_count(self) -> int` — `SELECT COUNT(*) FROM ssc`. Return 0 if table empty.

5. `insert_ssc_batch(self, records: list[SSCRecord], batch_size: int = 10000) -> tuple[int, int]` — batch insert SSC records. Use `INSERT OR IGNORE INTO ssc (smiles, atom_count, shift_list, avg_shift, min_shift, max_shift)`. Use `record.shift_list_as_json()` to serialize shift_list. Check `cursor.rowcount > 0` after each insert to distinguish inserted vs skipped (UNIQUE constraint on smiles). If inserted and `record.bitset is not None`, insert into `ssc_bitset` using `cursor.lastrowid`. Commit every `batch_size` records. Return `(inserted, skipped)`.

6. `iter_ssc_bitsets(self, batch_size: int = 100_000) -> Iterator[tuple[int, bytes]]` — select all `(ssc_id, bitset)` from `ssc_bitset ORDER BY ssc_id`. Use `cursor.fetchmany(batch_size)` in a while loop. Yield `(row["ssc_id"], bytes(row["bitset"]))`. Add `from __future__ import annotations` and `from typing import TYPE_CHECKING` with `if TYPE_CHECKING: from collections.abc import Iterator`.

7. `get_ssc_by_id(self, ssc_ids: list[int]) -> list[SSCRecord]` — return empty list if `ssc_ids` is empty. Build `IN (?, ?, ...)` placeholder query for `ssc` table. Select `id, smiles, atom_count, shift_list, avg_shift, min_shift, max_shift`. Construct `SSCRecord` from each row — the `field_validator` on `shift_list` will parse the JSON string automatically.

Import `sqlite3`, `Path`, and fragment-local schema/model imports. Do NOT import anything from the existing `database/` module — the fragment module must be fully independent.
  </action>
