  <action>
Perform a complete de novo CASE workflow on Ibuprofen using ONLY the thin CLI commands and the knowledge from skill documents. This is NOT a coding task — it's an execution task that validates the architecture works.

**Important context:**
- Ibuprofen: C13H18O2 (molecular formula)
- Data at: data/Ibuprofen/ (Bruker experiments in numbered subdirectories)
- Known structure: CC(C)Cc1ccc(cc1)C(C)C(=O)O
- Expected: Correct structure in top 3 after ranking

**Step-by-step workflow (following skill/CASE/SKILL.md):**

1. **Read available spectra** to understand what experiments exist:
   ```bash
   for d in data/Ibuprofen/*/; do lucy read 1d "$d" --format json 2>/dev/null || lucy read 2d "$d" --format json 2>/dev/null; done
   ```
   Identify which directories contain: 13C, DEPT-135, HSQC, HMBC, etc.

2. **Pick 13C peaks** (observe all carbon positions):
   ```bash
   lucy pick 1d data/Ibuprofen/<13C_dir> --format json
   ```

3. **Pick DEPT-135 peaks** (identify protonated carbons):
   ```bash
   lucy pick 1d data/Ibuprofen/<DEPT135_dir> --format json
   ```

4. **Pick raw HSQC peaks** (thin — no DEPT guidance):
   ```bash
   lucy pick hsqc data/Ibuprofen/<HSQC_dir> --format json
   ```

5. **Apply DEPT-guided filtering** (AI intelligence from skill/SKILL.md Section 3):
   - Match HSQC carbon positions to DEPT-135 peak positions (±1.0 ppm tolerance)
   - Extract multiplicity from DEPT-135 sign (positive = CH/CH3, negative = CH2)
   - Filter HSQC peaks to only those matching DEPT positions
   - Note quaternary carbons (in 13C but not in DEPT)

6. **Pick raw HMBC peaks** (thin — no cross-validation):
   ```bash
   lucy pick hmbc data/Ibuprofen/<HMBC_dir> --format json
   ```

7. **Apply HMBC cross-validation** (AI intelligence from skill/SKILL.md Section 3):
   - Filter HMBC peaks where carbon position matches known 13C positions (±1.5 ppm)
   - Filter HMBC peaks where proton position matches HSQC proton positions (±0.1 ppm)
   - Reject peaks that don't match both criteria

8. **Check symmetry**:
   ```bash
   lucy analyze symmetry C13H18O2 data/Ibuprofen/<13C_dir> --format json
   ```
   If observed < expected, note symmetry-equivalent atoms.

9. **Build LSD file** (AI writes directly using skill/SKILL.md Section 6 and skill/diagnostic/SKILL.md Section 1):
   - Write MULT lines for each atom (element, hybridization from shift region, H count from DEPT)
   - Write HSQC lines from filtered HSQC peaks
   - Write HMBC lines from validated HMBC peaks (start with high-confidence subset per incremental strategy)
   - Validate: sp2 count must be even, H budget must match formula
   - Write to a .lsd file

10. **Run LSD**:
    ```bash
    lucy lsd run ibuprofen.lsd --format json
    ```

11. **Convert solutions to SMILES** (if solutions found):
    ```bash
    outlsd 5 < ibuprofen.sol > ibuprofen_solutions.smi
    ```

12. **Rank solutions**:
    ```bash
    lucy lsd rank ibuprofen_solutions.smi --spectrum data/Ibuprofen/<13C_dir> --format json
    ```

13. **Check results**: Verify that Ibuprofen's canonical SMILES (or structural equivalent) appears in top 3.

**If LSD fails (0 solutions):** Apply diagnostic knowledge from skill/diagnostic/SKILL.md — check sp2 count, H budget, remove problematic HMBC correlations, retry. This is part of the validation — the AI should be able to self-diagnose and iterate.

**If too many solutions:** Add more HMBC correlations (incremental strategy from skill/SKILL.md Section 5).

Log all intermediate results and the final ranking in the summary.
