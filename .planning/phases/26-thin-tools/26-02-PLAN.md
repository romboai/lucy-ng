New signature:
```python
@pick.command("hsqc")
@click.argument("path", type=click.Path(exists=True))
@click.option("--threshold", "-t", type=float, default=0.05, help="Peak threshold (default: 0.05).")
@click.option("--format", "output_format", type=click.Choice(["text", "json"]), default="text")
def pick_hsqc(path: str, threshold: float, output_format: str) -> None:
    """Pick raw HSQC peaks above threshold.

    PATH is the path to the HSQC Bruker experiment directory.

    Returns raw cross-peaks without DEPT-guided filtering. The AI agent
    applies DEPT-guided filtering logic using skill knowledge.
    """
```

Implementation: Read 2D spectrum with BrukerReader.read_2d, call PeakPicker2D.pick_peaks(spectrum, threshold=threshold), format output as JSON or text. JSON output should include: experiment_type, count, peaks (f1_position, f2_position, intensity). Same pattern as existing `pick 2d` command.

BEFORE: Takes formula + hsqc_path + dept135_path, internally calls DEPTGuidedPicker.pick_hsqc_peaks() for DEPT-guided picking, then passes result to SymmetryAnalyzer.

AFTER: Takes formula + c13_path (a 1D 13C spectrum path), internally picks 1D 13C peaks with AdaptivePeakPicker, counts observed peaks, compares to expected carbons from formula. No HSQC, no DEPT, no intensity analysis.

New signature:
```python
@analyze.command("symmetry")
@click.argument("formula")
@click.argument("c13_path", type=click.Path(exists=True))
@click.option("--threshold", "-t", type=float, default=None, help="Peak threshold (auto if not set).")
@click.option("--format", "output_format", type=click.Choice(["text", "json"]), default="text")
def analyze_symmetry(formula: str, c13_path: str, threshold: float | None, output_format: str) -> None:
    """Compare observed 13C peaks with expected carbon count from formula.

    FORMULA is the molecular formula (e.g., C13H18O2).
    C13_PATH is the path to the 13C Bruker experiment.

    Returns raw comparison of observed signal count vs expected carbon count.
    The AI agent interprets the difference using symmetry detection knowledge.
    """
```

Implementation:
1. Read 1D spectrum with BrukerReader.read_1d(c13_path)
2. Pick peaks with AdaptivePeakPicker (pass threshold if provided)
3. Parse carbon count from molecular formula (extract 'C' count — use a simple regex: `re.search(r'C(\d+)', formula)`)
4. Output JSON: { formula, expected_carbons, observed_peaks, difference }
5. Output text: one-liner summary

Remove import of SymmetryAnalyzer and DEPTGuidedPicker from analyze.py. Add import of AdaptivePeakPicker and re.
