**src/lucy_ng/detection/models.py:**
Create Pydantic v2 models following the project convention (see database/models.py):

1. `HybridisationDistribution(BaseModel)`:
   - sp3: float = 0.0  # Frequency 0.0-1.0
   - sp2: float = 0.0
   - sp1: float = 0.0
   - Property `dominant` -> str: Returns the hybridisation state with highest frequency, or "unknown" if all zero
   - Property `is_definitive` -> bool: Returns True if one state has >95% frequency

2. `HybridisationResult(BaseModel)`:
   - shift_ppm: float  # Queried shift
   - window_ppm: float  # Query window used
   - radius: int  # HOSE radius used
   - threshold: float  # Minimum frequency threshold used
   - distribution: HybridisationDistribution  # Filtered distribution (states >= threshold)
   - total_observations: int  # Total sp3+sp2+sp1 count across all matching HOSE codes
   - unique_hose_codes: int  # Number of HOSE codes that contributed
   - has_data: bool  # False if no matching HOSE codes found or all hybridisation counts are 0
   - warning: str | None = None  # Warning message (e.g., "no hybridisation data - database needs regeneration")
   - Method `summary(self) -> str`: Human-readable text output like:
     ```
     Hybridisation Detection: 130.5 ppm (window +/- 2.0 ppm, radius 3)
     Distribution: sp2=85.2%, sp3=14.8% (sp1 excluded, <1%)
     Dominant: sp2 (definitive)
     Based on 12,450 observations from 342 HOSE codes
     ```
   - Method `model_dump_json(indent=2)`: Inherited from Pydantic BaseModel, used for --format json

**src/lucy_ng/detection/detector.py:**
1. `StatisticalDetector` class:
   - Constructor: `__init__(self, db_path: Path)` -- opens DatabaseManager as context manager
   - Uses `self._db = DatabaseManager(db_path)` and connects in __init__. Also implements __enter__/__exit__ for context manager usage.
   - Method `detect_hybridisation(self, shift_ppm: float, radius: int = 3, window_ppm: float = 2.0, threshold: float = 0.01) -> HybridisationResult`:
     a. Call `self._db.get_hose_stats_by_shift_window(shift_ppm, radius, window_ppm)`
     b. Aggregate sp3_count, sp2_count, sp1_count across all returned records (sum all)
     c. Compute total = sp3 + sp2 + sp1
     d. If total == 0: Return result with has_data=False and warning="No hybridisation data found. Database may need regeneration with v4 schema."
     e. Compute frequencies: sp3_freq = sp3/total, sp2_freq = sp2/total, sp1_freq = sp1/total
     f. Apply threshold filter: Set frequencies below threshold to 0.0
     g. Normalize remaining frequencies so they sum to 1.0 (or close to it)
     h. Return HybridisationResult with all fields populated
   - Method `close(self) -> None`: Closes database connection

The detector does NOT own any HOSE generation or RDKit logic -- it is purely a database query + frequency computation layer. This keeps it simple and testable.

```python
"""CLI commands for statistical detection of structural constraints."""

import json
from pathlib import Path

import click

from lucy_ng.database import DatabaseFinder


@click.group()
def detect() -> None:
    """Statistical detection of structural constraints from shifts."""
    pass


@detect.command("hybridisation")
@click.argument("shift_ppm", type=float)
@click.option(
    "--db", "-d", type=click.Path(exists=True), default=None,
    help="Path to SQLite database with HOSE stats.",
)
@click.option(
    "--radius", "-r", type=int, default=3,
    help="HOSE radius for query (default: 3).",
)
@click.option(
    "--window", "-w", type=float, default=2.0,
    help="Shift window in ppm (default: +/- 2.0).",
)
@click.option(
    "--threshold", "-t", type=float, default=0.01,
    help="Minimum frequency to include (default: 0.01 = 1%).",
)
@click.option(
    "--format", "output_format",
    type=click.Choice(["text", "json"]), default="text",
    help="Output format.",
)
def hybridisation_command(
    shift_ppm: float,
    db: str | None,
    radius: int,
    window: float,
    threshold: float,
    output_format: str,
) -> None:
    """Detect hybridisation state from 13C chemical shift.

    Queries the HOSE statistics database for all entries within a shift
    window and computes sp3/sp2/sp1 frequency distributions. States
    below the frequency threshold are excluded.

    Examples:

        lucy detect hybridisation 130.5

        lucy detect hybridisation 25.0 --radius 4 --window 1.5

        lucy detect hybridisation 155.0 --format json
    """
    from lucy_ng.detection import StatisticalDetector

    # Find database
    db_path = Path(db) if db else DatabaseFinder.find_hose_database()
    if not db_path:
        click.echo(
            "Error: No HOSE database found.\n\n"
            "Options:\n"
            "  1. Download database: lucy database download\n"
            "  2. Specify path: lucy detect hybridisation 130.5 --db /path/to/db",
            err=True,
        )
        raise SystemExit(1)

    # Run detection
    try:
        detector = StatisticalDetector(db_path)
        result = detector.detect_hybridisation(
            shift_ppm, radius=radius, window_ppm=window, threshold=threshold
        )
        detector.close()
    except Exception as e:
        click.echo(f"Error: {e}", err=True)
        raise SystemExit(1)

    # Output
    if output_format == "json":
        click.echo(result.model_dump_json(indent=2))
    else:
        click.echo(result.summary())

    # Warn if no data
    if result.warning:
        click.echo(f"\nWarning: {result.warning}", err=True)
```
