Create `src/lucy_ng/fragments/lsd_formatter.py` with a `DEFFFormatter` class containing all static methods. Follow the research-verified algorithm exactly:

**`smiles_to_fragment_content(smiles: str) -> str`:**
- Parse SMILES with `Chem.MolFromSmiles(smiles)`. Raise `ValueError` if None.
- First line: `; Fragment: {canonical_smiles}` (use `Chem.MolToSmiles(mol)`)
- For each atom in `mol.GetAtoms()`:
  - Index: `atom.GetIdx() + 1` (1-based for LSD)
  - Symbol: `atom.GetSymbol()`
  - Hybridization: if `atom.GetIsAromatic()` or `hyb == SP2` then `"2"`, elif `SP3` then `"3"`, elif `SP` then `"1"`, else `"(2 3)"` (fallback)
  - H count: `atom.GetTotalNumHs()`
  - Write: `SSTR S{idx} {symbol} {lsd_hyb} {nh}`
- For each bond in `mol.GetBonds()`:
  - `LINK S{begin+1} S{end+1}`
- Return joined with newlines + trailing newline.

**`fragment_filename(smiles: str) -> str`:**
- Canonicalise: `Chem.MolToSmiles(Chem.MolFromSmiles(smiles))`
- Hash: `hashlib.sha256(canonical.encode()).hexdigest()[:12]`
- Return: `f"fragment_{hash_prefix}.lsd"`

**`write_fragment_file(smiles: str, output_dir: Path | None = None) -> Path`:**
- Generate content via `smiles_to_fragment_content`.
- Generate filename via `fragment_filename`.
- If `output_dir` is None, use `Path.cwd()`.
- Write content to `output_dir / filename`.
- Return the full path.

**`deff_command(fragment_number: int, filepath: str) -> str`:**
- Return `f'DEFF F{fragment_number} "{filepath}"'` -- MUST use double quotes per LSD 3.4.9 requirement.

**`fexp_command(fragment_numbers: list[int], logic: str = "OR") -> str`:**
- Empty list: return `""`.
- logic="NOT" with single fragment: return `f'FEXP "NOT F{n}"'`.
- Single fragment: return `f'FEXP "F{n}"'`.
- Multiple with OR: return `f'FEXP "F1 OR F2 OR ..."'`.
- Multiple with AND: return `f'FEXP "F1 AND F2 AND ..."'`.

Type hints: use `from __future__ import annotations`, `from pathlib import Path`. Imports: `hashlib`, `from rdkit import Chem`, `from rdkit.Chem import Hybridization`.
