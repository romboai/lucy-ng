**Add `to-lsd` CLI command:**

Add a new Click command `to_lsd` (exposed as `to-lsd`) on the `fragment` group. Import `DEFFFormatter` from `lucy_ng.fragments.lsd_formatter`.

Arguments:
- `smiles` (required, type=str): Fragment SMILES string.

Options:
- `--output-dir` (type=click.Path(path_type=Path), default=None): Directory to write fragment file. If None, writes to CWD.
- `--format` / `output_format` (choice: text/json, default="json"): Output format.

Implementation:
1. Call `DEFFFormatter.write_fragment_file(smiles, output_dir=output_dir)` to write the .lsd file.
2. Generate DEFF command: `DEFFFormatter.deff_command(1, written_path.name)` (bare filename, not full path -- LSD resolves relative to CWD).
3. Generate FEXP command: `DEFFFormatter.fexp_command([1])`.
4. Read file content from the written path.

JSON output:
```json
{
  "smiles": "<input smiles>",
  "canonical": "<canonical smiles>",
  "filename": "<written filename>",
  "path": "<full path as string>",
  "deff_command": "DEFF F1 \"<filename>\"",
  "fexp_command": "FEXP \"F1\"",
  "atom_count": <N>,
  "content": "<file content>"
}
```

Text output:
```
Fragment file written: <path>
Atoms: <N>
DEFF command: DEFF F1 "<filename>"
FEXP command: FEXP "F1"
```

Error handling: Catch `ValueError` from DEFFFormatter (invalid SMILES), print error to stderr, raise `click.Abort()`.

Run: `ruff check src/lucy_ng/cli/fragment.py` and `mypy src/lucy_ng/cli/fragment.py` to verify clean.
