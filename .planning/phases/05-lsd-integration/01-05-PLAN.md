# Phase 5: LSD Integration — Plan

## Objective

Integrate the LSD (Logic for Structure Determination) solver to generate candidate molecular structures from NMR peak data. This phase creates the bridge between peak picking results and structure elucidation.

## Context

### LSD Overview

LSD is a constraint-based structure elucidation tool that takes:
- **Atom definitions** (MULT): element, hybridization, H count
- **Direct correlations** (HSQC): C-H one-bond connections
- **Long-range correlations** (HMBC): C-H 2-3 bond connections
- **H-H correlations** (COSY): through-bond H-H connections

And outputs candidate molecular structures as SMILES/MOL files.

### LSD Input File Format

```
; Atom definitions: MULT atom_num element hybridization h_count [charge]
MULT 1 C 2 0    ; sp2 carbon, 0 hydrogens (quaternary)
MULT 2 C 3 2    ; sp3 carbon, 2 hydrogens (CH2)
MULT 3 C 3 3    ; sp3 carbon, 3 hydrogens (CH3)

; Correlations
HSQC 2 2        ; atom 2 has direct H correlation at position 2
HMBC 1 3 2 3    ; atom 1 correlates to H3 through 2-3 bonds
COSY 2 3        ; H2 correlates to H3

; Chemical shifts (optional)
SHIX 1 129.5    ; 13C shift
SHIH 2 2.35     ; 1H shift
```

### Research Findings

From [LSD software documentation](https://nuzillard.github.io/LSD/index_ENG.html):
- LSD requires unambiguous atom status (element, hybridization, H count)
- HMBC correlations can specify bond distance bounds (2-3 by default)
- Solutions are output as numbered structure files

From [pyLSD](https://nuzillard.github.io/PyLSD/INSTALL.html):
- pyLSD wraps LSD with Python and adds 13C shift prediction ranking
- Requires Java for shift prediction component
- We'll target basic LSD first, pyLSD ranking later

### Available Data from Previous Phases

- `PeakList1D` (13C peaks with positions)
- `PeakList2D` (HSQC, HMBC, COSY correlations)
- `DEPTGuidedResult` (validated HSQC peaks + multiplicities: CH, CH2, CH3)
- Molecular formula (from user input or external source)

## Decision

**Target LSD first, then pyLSD.** LSD is simpler (no Java dependency) and pyLSD builds on it. This allows incremental development.

class LSDInputGenerator:
    """Generate LSD input files from peak data."""

    @staticmethod
    def generate(problem: LSDProblem) -> str:
        """Generate LSD input file content."""
        lines = []

        # Header comment
        lines.append(f"; LSD input for {problem.name}")
        lines.append(f"; Generated by lucy-ng")
        lines.append("")

        # Atom definitions (MULT)
        for atom in problem.atoms:
            lines.append(f"MULT {atom.index} {atom.element} "
                        f"{atom.hybridization.value} {atom.hydrogen_count}")

        # Correlations
        for corr in problem.correlations:
            if corr.correlation_type == "HSQC":
                lines.append(f"HSQC {corr.atom1_index} {corr.atom2_index}")
            elif corr.correlation_type == "HMBC":
                lines.append(f"HMBC {corr.atom1_index} {corr.atom2_index} "
                            f"{corr.min_bonds} {corr.max_bonds}")
            elif corr.correlation_type == "COSY":
                lines.append(f"COSY {corr.atom1_index} {corr.atom2_index}")

        lines.append("EXIT")
