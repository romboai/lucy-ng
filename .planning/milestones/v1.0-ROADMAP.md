## Milestone Summary

**Key Decisions:**
- nmrglue for NMR parsing (Most mature, BSD licensed, native Bruker support)
- Pydantic v2 for models (Type safety, validation, JSON serialization)
- Click CLI framework (Simpler than Typer, no extra dependencies)
- DEPT-guided adaptive thresholding (Lower HSQC threshold until all DEPT carbons matched)
- HMBC-guided peak picking (Filter by requiring C match in 13C/DEPT and H match in HSQC)
- N:1 shift matching for ranking (Handles molecular symmetry correctly)
- DOI-based data fetching (Parse NMRXiv DOIs directly for project/study IDs)

**Issues Resolved:**
- HSQC noise filtering via DEPT validation
- HMBC artifact removal via reference peak filtering
- Molecular symmetry handling in LSD input generation
- Solution ranking accounting for equivalent atoms

**Issues Deferred:**
- COSY correlation support in LSD constraints
- Stereochemistry handling (E/Z, R/S)

**Technical Debt Incurred:**
- None significant
